Typora 是一款 Markdown 编辑器和阅读器，同时支持Markdown和富文本，无论哪一种，都能做到所见即所得。

使用这款软件能够帮助用户轻松将文本转换到HTML，软件从底层向上设计，软件支持markdown的标准语法，同时这款软件还支持动态预览功能，一键预览，让一切都变得如此干净、纯粹，是一款不可多得的优质markdown编辑器。

  • 主题：Typora 为您导出文档时提供了多种主题。 如果您了解 CSS，则可以自定义这些主题。 打开“首选项”窗口并查看“外观”>“主题”下的设置。

  • 严格模式：Typora 为希望对标题、有序列表和无序列表实施语法限制的用户提供了严格的模式设置。 例如，您可以将无序列表配置为仅使用连字符而不使用星号。 在 Markdown > Syntax Preference 下的 Preferences 窗口中配置这些设置。 有关其他信息，请参阅 Typora 文档。

  • 源代码模式：您可以通过选择 View > Source Code Mode 来禁用 Typora 的实时编辑器。 这将显示实时编辑器隐藏的所有 Markdown 格式。

  • 导出选项：当您准备好发布 Markdown 文档时，Typora 在文件 > 导出下提供了多种导出选项。 一些导出选项，如 Microsoft Word 和 LaTeX 格式，需要 Pandoc。

  • 扩展语法：Typora 使用的是 GitHub 风格的 Markdown 语法，扩展了任务列表、表格、表情符号、数学公式、代码高亮等常用功能

  • 快捷操作：Typora 对几乎所有 Markdown 语法都提供了快捷操作，通过菜单栏的 Paragraph 和 Format 中的功能可以快速设置标记语法，一些常用的操作都有快捷键，用起来非常高效。

下载安装

Typora官方网址是：https://typora.io/，可以看到Typora的介绍，也有一个中国网址：https://typoraio.cn/，不建议使用，国内代理，你懂的。

下载官方最新版：所有版本

安装完成后，先不要急于打开Typora，给软件打上补丁后再打开。

目前关于Typora有两种补丁方式，一种是使用 winmm.dll 补丁，一种是使用 app.asar 补丁，两者都简单有效。下面我们一一尝试。

使用winmm.dll

winmm.dll 补丁目前适用 Typora1.4.X，使用简单，只需要将 winmm.dll 文件复制到Typora安装目录便可。

使用app.asar

app.asar 补丁需要跟着版本来使用，目前最新版是1.4.7，再往高了可能会出问题。复制app.asar到安装目录下的resources下替换原文件即可。
